# ODPlatform

## Current Status

This repository contains the D2 foundation for ODPlatform: a monorepo layout, a
Python package under `apps/platform`, shared path constants, logging utilities,
system snapshots, timing helpers, and a project initialization CLI.

## Purpose

ODPlatform is a target-detection platform skeleton. This stage focuses on the
engineering base that later data, training, evaluation, and inference modules
will reuse.

## Future Plan

Future milestones will add configuration management, data conversion, training,
evaluation, inference, and additional app entry points such as web and desktop
clients.
